### Getting Started with Task Management App (Frontend)

- This is the frontend of Task Management app.
- This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

### Project setup

- Clone the repository on your machine
- cd frontend
- npm install
- npm run start

Note: Ensure that Backend server is up and it's URL is updated in the `frontend/constants.js` file
